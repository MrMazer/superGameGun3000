# GraviGun

**Автор:** Крутых Роман Вячеславович  
**Группа:** М8О-407Б-22

## О проекте

**GraviGun** — 3D-бродилка от первого лица на Unity. Игрок оказывается на заброшенной локации, где для продвижения нужно использовать гравитационную пушку: взаимодействовать с физическими объектами окружения и решать пространственные задачи, чтобы добраться до выхода.

## Суть игры

Исследуйте уровень, находите и активируйте гравитационную пушку, манипулируйте предметами среды и проходите короткую цепочку задач — от установки пушки на платформу до постройки прохода из объектов и открытия выхода.

## Механики

### 1. Гравитационный захват объектов

Игрок подходит к стационарной гравитационной пушке и через неё захватывает физические предметы на расстоянии (ящики, обломки), переносит их в нужную точку и сбрасывает с импульсом.

- Захват и удержание объектов лучом из ствола пушки  
- Регулировка дистанции удержания  
- Бросок с импульсом или мягкое отпускание  

### 2. Постройка лестницы из объектов

Используя гравитационную пушку, игрок выстраивает из захваченных предметов лестницу или платформу, чтобы преодолеть препятствие и добраться до недоступной зоны уровня.

- Фиксация объекта в заданной точке (snap)  
- Подсветка интерактивных зон и целей  
- Кнопки и ворота для финального прохождения уровня  

## Используемые инструменты Unity

| Инструмент | Назначение |
|------------|------------|
| **Cinemachine** | Камера при работе с гравитационной пушкой |
| **Bake Lighting** | Запечённое освещение сцены |
| **VFX Graph** | Эффекты гравитационного захвата и подсветка интерактивных объектов |
| **Post-processing** | Общая атмосфера и визуальная обработка кадра |

## Управление (кратко)

| Действие | Клавиша |
|----------|---------|
| Движение | `W` `A` `S` `D` |
| Прыжок | `Пробел` |
| Взаимодействие | `E` |
| Захват / бросок (на пушке) | ЛКМ |
| Отпустить объект | ПКМ |
| Дистанция удержания | Колёсико мыши |

## Запуск

1. Откройте папку проекта в **Unity** (версия, с которой собирался проект).  
2. Откройте сцену уровня в `Assets/`.  
3. Нажмите **Play**.

## Структура кода

Основная логика находится в `Assets/Scripts/`:

- `GravityGun.cs` — гравитационная пушка  
- `PlayerController.cs` — управление игроком  
- `LevelManager.cs` — цели и прогресс уровня  
- `SnapPoint.cs`, `GateController.cs`, `ButtonInteract.cs` — головоломки уровня  
- `UIManager.cs` — интерфейс и подсказки
